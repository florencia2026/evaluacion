Desarrollé un proyecto de "Librería Carrusel" donde se encuentran diversos libros a la venta.
La página es para que hayan más ventas online y que el cliente pueda informarse adecuadamente del libro disponible, además de tener la posibilidad de hacer otro tipo de consultas.

Las carpetas usadas con de css para estilos, img para las imágenes, index para el código principal html y readme para escribir las descripciones.

Utilicé prompts para crear de a poco el código que deseaba para la página que tenía pensada. 
Primero hice las conexiones entre css y html, usando un prompt claro para la ia.
Luego expliqué cómo deseaba que se viera la página, partiendo por el título y subtítulos requeridos, teniendo en cuenta los archivos de aai.
Luego agregaba botones, explicando claramente lo que quería.
Después agregué imágenes que descargué previamente, creando la carpeta img y poniéndolas ahí, las imágenes las puse a mano.
Finalmente le pedí perfeccionar el estilo de la página para que se viera más profesional.
Escribí muchos prompts más complejos y otros más simples, pero todos me sirvieron para lograr este resultado.
